Search Page

Search is simply a web page that features an interactive search bar that is supposed to resemble the Google search bar. It is the first web page I learned to built and is more basic compared to the other pages.

Loading Page

Loading features an increasing percentage that is continuously becoming blurrier as the selected image is gradually becoming clearer. This page is supposed to resemble an image waiting to fully load on a screen.

Practice Page

This page was solely designed as a workspace to practice my skills at HTML, CSS, and Java Script. It features a blue square continously increasing and decreasing in size.

Expanding Page

Expanding includes five images available for users to tap and see the picture expand with a caption to read. This is a more complex and interactive page.

Event Page

This is a web page that is more interactive and incorporates five different "buttons" that users could click. Each button does something different.

Interactive Web Page

Interactive Web Page is another interactive one that allows the user to change the background with the push of a button.

Sound Page

Sound features six different noises that become available to hear with just the push of a button. This is a more complex page in the sense that it allows for the incorporation of various sounds to the web page.

Nav Page

Nav serves to resemble a template or sample website page. It includes basic structures like headings, images, and paragraphs that one would typically see in a website, and it is also somewhat interactive when scrolling or hovering.

Insect Game

Insect is not only a web page, but it also is an interactive game for users to play. It tests user's patience against insects in a fun way.
